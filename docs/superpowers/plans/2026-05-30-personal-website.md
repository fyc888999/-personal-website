# 个人网站实施计划

**目标：** 创建一个现代、响应式的个人网站，展示个人信息、游戏项目和联系方式。

**架构：** 单页应用风格，使用语义化 HTML5、CSS3（Flexbox + Grid）和原生 JavaScript。所有代码内嵌在单个 HTML 文件中。

**技术栈：** HTML5、CSS3、JavaScript（ES6+）、frontend-design 技能
