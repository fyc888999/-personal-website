**项目：** 符奕宸的个人主页

## 1. 项目概述

### 1.1 目标
创建一个现代、简洁、响应式的个人网站，展示个人信息、项目作品和联系方式。

### 1.2 目标用户
- 潜在雇主和合作伙伴
- 同学和老师
- 对项目感兴趣的访客

### 1.3 核心需求
- 单页应用（SPA）风格
- 平滑滚动导航
- 响应式设计（手机、平板、电脑）
- 无外部依赖（纯 HTML/CSS/JS）
- 暗色渐变主题

## 4. 动画规范

### 4.1 页面加载动画
- 元素依次淡入上滑
- 延迟：0.1s - 0.5s
- 持续时间：0.6s
- 缓动函数：`ease-out`

### 4.2 滚动动画
- 元素进入视口时触发动画
- 使用 Intersection Observer API
- 动画：淡入 + 上滑 30px

### 4.3 交互动画
- 按钮悬停：`transform: translateY(-3px)` + 阴影
- 卡片悬停：`transform: translateY(-10px)` + 阴影
- 链接悬停：颜色渐变 + 下划线展开

### 4.4 导航栏动画
- 显示/隐藏：`transform: translateY()` 过渡
- 持续时间：0.3s
- 缓动函数：`ease-in-out`

### 6.1 文件结构
```

├── index.html          # 主页面（新文件）
├── guess.html          # 猜数字游戏（已有）

└── docs/
    └── superpowers/
        └── specs/
            └── 2026-05-30-personal-website-design.md
```

### 6.2 技术栈
- HTML5
- CSS3（Flexbox + Grid）
- JavaScript（ES6+）
- 无外部库/框架
- **设计工具：** frontend-design 技能（将安装）

### 6.3 关键技术点
- **平滑滚动：** `scroll-behavior: smooth`
- **智能导航：** Intersection Observer + 滚动方向检测
- **横向滚动：** 鼠标拖拽 + 触摸滑动
- **动画性能：** `transform` 和 `opacity` 优化
- **响应式图片：** `srcset` 和 `sizes` 属性

### 6.4 性能优化
- CSS 和 JS 内联（单文件）
- 图片懒加载
- 动画使用 `requestAnimationFrame`
- 避免强制同步布局
